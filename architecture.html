System Architecture

How everything connects. Control → Value → Participation → Liquidity.

TCJ Smart Volume Concept operates as a structured sequence. Each component depends on the previous one. When the order breaks, structure breaks.

The framework follows this progression:

Control (TCC) → Value (AEV) → Participation (VDI) → Liquidity (SCC or TCC)

This is not theory. It is the order in which dominance unfolds.

1. Control Initiates the Move

A Tactical Control Candle is the starting event. It marks the moment initiative appears. Without a control event, there is no structural reason to expect continuation.

Control is the trigger that activates the rest of the framework.

2. Value Defines the Reference

Once control appears, value resets. AEV forms immediately from that control event and becomes the reference for the dominant side.

Now the question becomes: Is price holding relative to that value, or failing to defend it?

  • If value is respected, the control event remains structurally valid.
  • If value is violated, control weakens.

Control starts the move. Value determines if it survives.

3. Participation Validates Continuation

After control establishes value, participation must expand. VDI measures whether the dominant side is still pressing with real imbalance.

This is where many moves fail. Price may drift. But without participation expansion, continuation lacks structural support.

Participation is what transforms a control event into sustained movement.

4. Liquidity Becomes the Objective

When control holds, value is respected, and participation expands, price rotates toward liquidity.

Liquidity often sits at:

  • Prior Structural Control Candles
  • Opposing Tactical Control zones

These areas represent unfinished business — prior dominance, trapped positioning, or unresolved imbalance.

Liquidity is not random. It is where structure naturally completes.

How It Connects

  • If control forms but value fails → structure weakens.
  • If value holds but participation does not expand → movement stalls.
  • If participation expands without control foundation → fragility increases.

Every phase depends on alignment. The architecture forces sequencing:

  • No control → no value reset.
  • No value respect → no structural validity.
  • No participation → no continuation.
  • No continuation → liquidity remains unreachable.

The strength of the framework is not in any single component. It is in the order.
