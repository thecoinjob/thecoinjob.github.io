TCJ Smart Volume Concept

TCJ Smart Volume Concept is a structural crypto futures framework built on control, value reset, and participation quality. It is not indicator-based trading. It is not prediction-driven analysis. It is a dominance model.

The system is built from clearly defined components. Each serves a specific purpose. Together, they form a structured decision framework.

Core Components

1. Control Candles (CC)

Control Candles represent moments where market dominance visibly shifts. Not every candle matters. A Control Candle marks a change in initiative — where either aggression or absorption establishes temporary or structural control.

There are two primary classifications:

  • Structural Control Candles (SCC): These define broader regime context. They anchor swing structure and establish dominant direction zones.
  • Tactical Control Candles (TCC): These represent localized control shifts. They define actionable dominance inside the broader structure.

Control is the foundation of the entire framework. Without control, there is no valid value reset.

2. Anchored Execution Value (AEV)

When control shifts, value resets. Anchored Execution Value is derived from a Control Candle and represents the new fair value of price under that dominance.

Price interaction with AEV reveals acceptance or rejection:

  • Sustained positioning relative to AEV suggests acceptance of control.
  • Failure to hold relative positioning suggests rejection.

AEV is not a moving average. It is a structural reference tied directly to control events.

3. Volume Delta Imbalance (VDI)

VDI measures directional participation quality. Price can move without real commitment. VDI distinguishes between fragile movement and sustained continuation.

It operates across layered horizons:

  • Execution layer
  • Commitment layer
  • Regime layer

Alignment across layers suggests trend health. Divergence signals instability. VDI validates strength — it does not predict it.

4. Structural Validation

Beyond control and participation, the framework considers:

  • Trend alignment
  • Volume expansion
  • Liquidity positioning

Price must align structurally, not emotionally. Continuation is only favored when control, value, and participation operate in agreement.
